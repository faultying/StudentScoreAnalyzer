# 学生成绩统计系统 (Student Score Analyzer)

## 项目简介
这是一个基于 Python 的数据分析小工具，旨在帮助教师或教务人员快速处理学生考试成绩。程序能够自动读取数据文件，计算各项统计指标（如平均分、方差、绩点等），并生成直观的可视化图表。

## 主要功能
- **多格式支持**：支持导入 `.txt` 和 `.csv` 格式的成绩单。
- **统计分析**：
  - 计算各科目的最高分、最低分、平均分。
  - 统计各分数段（优秀、良好、及格、不及格）的人数及占比。
- **个人评估**：
  - 自动计算每位学生的平均绩点 (GPA)。
  - 计算学生成绩的方差与标准差，分析成绩稳定性。
- **数据可视化**：
  - 使用 `matplotlib` 自动生成成绩分布的柱状图和饼状图。

## 环境要求
Python 3.6+
依赖库：`pandas`, `matplotlib`

## TXT 格式：
每行一个数字，代表一个分数。
## CSV 格式：
第一列为学号，后面每列为一科成绩。
